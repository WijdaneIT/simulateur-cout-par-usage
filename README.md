# 👗 Cost-per-Wear Predictor — Rentabilité réelle d'un achat vestimentaire

## Contexte
Ce projet est né à la bibliothèque de la mairie de Paris, en face du BHV, 
pendant la polémique de l'arrivée de Shein dans cette institution de la mode parisienne.
Face au débat "fast fashion pas chère vs vêtements de qualité", 
j'ai voulu répondre par les données : est-ce qu'acheter moins cher revient vraiment moins cher ?

## Problème résolu
Calculer le **coût réel par utilisation** d'un vêtement en fonction de son prix, 
sa durabilité estimée et sa fréquence de port — et prouver que le prix d'achat 
n'est pas un bon indicateur de rentabilité.

## Stack technique
- Python · Pandas · Random Forest · Streamlit

## Résultat
Application interactive permettant de comparer deux vêtements 
et de visualiser lequel est réellement le plus rentable sur le long terme.
